## Type
- `feat` 새로운 기능 추가
- `fix` 버그 수정
- `docs` 문서 수정
- `style` 코드 포맷팅, 띄어쓰기, 들여쓰기
- `chore` (코드 수정 없이) 빌드 스크립트 설정 변경, 패키지 매니저 수정
- `test` 테스트 코드, 리팩토링 테스트 코드 추가
- `refactor` 코드 리팩토링
- `ci` ci 관련 스크립트 파일을 수정시
- `merge` merge 시 사용
